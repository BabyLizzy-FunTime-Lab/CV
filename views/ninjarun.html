Ninjarun

Small intro

This game is still in development.
I build it with the help of a manual published by CoderDojo purely as a hobby. The book I used has some information that has been depricated and it's writen for complete beginners. Because of this, my game isn't an exact replica of what is in the book. Further more, the book did not use the full potential of object oriented programming. For example, the main character of the game isn't writen as an object, even though it should be. The book also doesn't explain how to launch the game as a node.js webapp.

I am glad I took the time to write this simple game because it thaught me new concepts in animation. I will defenatly expand this game. First by cleaning up the code to make it more efficient and readable and then by including audio, levels, a score system, different enemies, character selection and bosses. Optimizing the game for mobile platforms is also in the to-do list.
