Real Estate Care

This is my front-end frameworks final project for the LOI. It's a mobile first web app made for a fictitious real estate inspection company.

The App is made with ionic/vue. State management is done with Pinia and api calls are made with Axios. This app is a demo. Uploaded Image files and text data are temporally stored as text on my Heroku My JSON Server.

Please view on mobile for best experience.

Leden Administratie

This was my PHP & SQL / Laravel final project for the LOI. The app is made in Laravel and has a MySQL database. This was mostly a backend project.
The app is hosted on my Heroku account.

The app is a "user membership management" (leden adminstratie) application where new users can be made and a subscription fee is calculated based on the age of the user and membership plans. The users are grouped into families.
