My Projects

While studying at The NewYork Code Academy and The LOI I made plenty of practice websites. The projects listed here are my best looking and functioning projects.

Gitaar

This was my HTML5 & CSS3 endproject for the LOI course I am following. I love gitaars so I had a lot fun working on this project. I passed the coures with this project.

As you can imagine I was only allowed to use HTML5 & CSS3 to build the entire website. I was only allowed a little JS to add a google maps feature in the contacts page. The responsiveness of the page is done with the flexbox.

My plans for improvement are to improve the maps functionality and to add a login backed by a user database.

Movie Buff

A friend working at Sogeti gave me this project to help me test my own skills and help me get a feel of how far I've come. I'm pleased with my progress and my friend gave me some good tips for improvement.

This website is build in pure JavaScript(ES6), HTML5 and CSS3. It sends queries to a movie data base and I used promises to make up for the response delay.

My plans for improvement are to include a login so I can keep track of searches and personalise the start page of the user.

Ninjarun

This is a sidescolling endless runner game. You play as a ninja avoiding incomming robots and obsticals. The game is written in pure JavaScript, as a full-on frontend game. The entire game and it's resouces get served to the user.

This game had alots of room for expansion. I plan on adding music, soundeffects, a character selection, flying enemies and an end boss. It's also the first game I have ever written in an objectoriented programming language.
