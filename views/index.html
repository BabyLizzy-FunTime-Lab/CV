My Projects

While studying at The NewYork Code Academy and The LOI I made plenty of practice websites. The projects listed here are my best looking and functioning projects.

Gitaar

This was my HTML5 & CSS3 final project for the LOI course I am following. I love guitars so I had a lot fun working on this project. I passed the courese.

As you can imagine I was only allowed to use HTML5 & CSS3 to build the entire website. I was only allowed a little JS to add a google maps feature in the contacts page. The responsiveness of the page is done with the flexbox.

My plans for improvement are to improve the maps functionality and to add a login backed by a user database.

Movie Buff

A friend working at Sogeti gave me this project to help me test my own skills and help me get a feel of how far I've come. I'm pleased with my progress and my friend gave me some good tips for improvement.

I build this website in pure JavaScript(ES6), HTML5 and CSS3 to show that I have a good grasp on the essentials of front-end web development. My app queries an opensource movie database.

My plans for improvement are to use React.js to render the information.

Ninjarun

This is a sidescolling endless runner game. You play as a ninja avoiding incomming killerrobots!! The game is written in pure JavaScript, as a full-on front-end game. The entire game and it's resouces get served to the user.

This game has lots of room for expansion. I plan on adding music, soundeffects, a character selection, flying enemies and an end boss. It's also the first game I have ever written in an object oriented programming language.
